## API (feed)

| Метод | Путь                   | Назначение                 | Параметры                 | Ответ |
|:-----:|:-----------------------|:---------------------------|:--------------------------|:------|
| GET   | /api/posts/            | Following feed (по умолч.) | ?limit&after&filter=following| 200 { posts:[...], next_cursor }
| GET   | /api/posts/            | Общая лента                | ?limit&after&filter=all   | 200 { posts:[...], next_cursor }
